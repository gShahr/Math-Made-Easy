Module 2: Printing to the Console

  1. Add Standard Input/Output Library header at the beginning of your source code. This is required, so that later you can use the printf function. 
                            #include
  2. The main function is the starting point of every C program. 
                            #include  
                            int main(int argc, char* argv[]) { 
                        }
  3. Add the return/ending of the function inside the main function. 
                            #include 

                        int main(int argc, char* argv[]) {
                                return 0;
                        }
  4. Put a printf function above the return statement, to output a string to the console. 
                            #include 

                        int main(int argc, char* argv[]) {
                                printf("Hello World!");
                                return 0;
                        }
